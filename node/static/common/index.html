以下是common逻辑的实现清单

  • 添加html中lang属性
  • A标签去噪
    • 又有img又有文字，且文字内容跟img的alt一致时，取掉img的alt，以免重复读屏demo
    • 链接有title时，链接下的图片均可视为装饰信息，将img的alt取掉，以免读取图片信息demo
    • 链接没有title时，取链接下img的alt（第一个有效值）作为连接title，同时取掉下面所有img的altdemo
    • 既没有title也没有内部文字时，提供文字提示:这是一个指向XXX的链接demo
    • 既有title又有文字且信息一致时，去除链接titledemo
  • 处理img
    • 图片高度或宽度小于8像素的，将alt清空，设置aria-hidden为truedemo
  • 事件响应类
    • 移除页面上onfocus=this.blur()，保留其它onfocus事件demo
    • 检查页面上div，p，li，td，如果其有设置onclick事件，则为其添加tabindex使其可focus，并添加keypress事件，当用户输入enter时出发onclick事件demo
    • 将页面上的a，input，button，img，textarea的鼠标事件全迁移到焦点或输入事件，对应关系是：onmouseover -> onfocus; onmouseout -> onblur;onmousedown -> onkeydown;onmouseup -> onkeyupdemo
